# Hosted Policy Drafts

These drafts are for the managed hosted service. They are product/operator
drafts, not legal advice. Replace placeholders, review with the right legal/tax
advice for your jurisdiction, and publish only after the launch checklist is
complete.

## Drafts

- [`pricing-and-terms.md`](./pricing-and-terms.md): pricing, billing,
  cancellation, refund, tax, and launch caveats
- [`acceptable-use.md`](./acceptable-use.md): public acceptable-use and abuse
  reporting language
- [`privacy-and-data-requests.md`](./privacy-and-data-requests.md): hosted
  privacy and data request language
- [`launch-checklist.md`](./launch-checklist.md): final public policy and live
  billing readiness checklist
- [`realization-plan.md`](./realization-plan.md): implementation plan for admin,
  localization, observability, and source strategy

## Required Public Contacts

Before public signup, replace these placeholders everywhere:

- `<abuse@example.com>` for abuse reports and shared-domain reputation issues
- `<support@example.com>` for billing and general support
- `<privacy@example.com>` for privacy, export, and erasure requests

Do not publish the drafts while placeholders remain.
